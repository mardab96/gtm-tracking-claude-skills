# Output Standard

Every skill output should be:

1. **Evidence-first.** State what the data shows before interpreting it.
2. **Honest about missing data.** If an input is missing, say so and explain what it changes.
3. **Decision-oriented.** End with a clear next action or approval ask.
4. **Conservative with claims.** Confidence levels: high, medium, low, unknown.
5. **Human-in-the-loop.** Skills diagnose and recommend. They do not execute changes.
